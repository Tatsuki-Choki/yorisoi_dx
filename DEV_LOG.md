# 開発記録（ヨリソイDX LP）

## 概要
- 目的: 中小企業の意思決定者向けに、AI導入の相談・資料請求を獲得するLPを整備。
- 方針: Notion系のフラットデザイン、淡い水色をアクセントに採用。文章は訪問者ベネフィット優先。
- 主導線: 資料請求（送信後PDFダウンロード）を最重要CTAに統一。

## ファイル構成
- `index.html` : LP本体
- `company/index.html` : 会社概要ページ
- `privacy/index.html` : プライバシーポリシー
- `assets/yorisoi-dx-brochure.pdf` : 資料PDFのプレイスホルダー

## 要件・決定事項
- ブランド表記: 「ヨリソイDX」
- ターゲット: 中小企業の決裁者
- トーン: 信頼重視 / 先進的 / 読みやすさ優先
- 色: 淡い水色系のアクセント
- 会社情報: LPフッターから移動し、別ページ（会社概要）に集約
- 連絡先メール: `info＠tsukichiyo.jp`（全角＠。注記「送信時は半角に」）
- CTA: 資料請求を主導線
- 画像: 実画像は未導入。プレイスホルダーに適切な画像内容を明記
- 実績セクション: なし
- フォーム項目: 会社名・氏名・メール・相談内容（必須）／役職・従業員規模・電話（任意）
- 送信後: PDF自動ダウンロード表示。送信内容は将来的にDiscord通知予定

## LPのUI/UX調整ポイント
- ヒーロー: H1を「AIで、業務をもっとシンプルに。」に設定
- ヒーロー要点: 3カラム化して「迷わず始められる／費用の見通し／定着」
- ベネフィット訴求: 「何をするか」ではなく「何が得られるか」に全体を修正
- 料金表示: 表形式に変更し、税別をメイン・税込を括弧で小さく表示に統一

## 料金表示ルール
- 無料相談: 税別0円 / 税込0円
- AIツール開発: 税別50万円 / 税込55万円
- AI研修: 1〜5名 税別30万円 / 税込33万円
- AI研修: 6〜9名 税別25万円 / 税込27.5万円
- AI研修: 10名以上 税別20万円 / 税込22万円
- 「標準」「要件により変動」などの表現は削除
- 追加プラン案内を表の下に追加

## 追加・変更した主要セクション
- 会社概要ページを新規作成
  - 項目: 会社名 / 代表 / 所在地 / 連絡先 / 事業内容 / 設立 / 資本金
  - 事業内容・設立・資本金は枠のみ（空欄）
- プライバシーポリシーページを新規作成

## 画像プレイスホルダー配置
- ヒーロー: 「中小企業のオフィス/会議風景（人物あり・自然光・清潔感）」
- ベネフィット直下: 「業務フローのイメージ（抽象的な図/作業中の手元）」
- 資料請求カード: 「資料PDFの表紙モック（タイトル＋淡い水色）」

## 未対応 / 次に引き継ぐ項目
- 画像の本差し替え（Unsplashなどを使う場合はAPIキー・帰属表記に注意）
- Discord通知用フォーム送信の中継（GitHub Actions等）
- 資料PDF本体の差し替え（`assets/yorisoi-dx-brochure.pdf`）
- 文言の最終調整（必要であれば短文化）

## 備考
- 画像APIを使う場合は、UnsplashのAPI利用規約に従うこと（帰属表記・hotlink必須）。
- 会社情報はLPから外しているため、フッターリンク経由で参照する運用。
